Open-electronics RFTide shield libraries for Arduino
Version: 1.0.0
Release Date: 2013-08-25
http://www.open-electronics.org/

Summary
This is a libraries for the Arduino that interfaces with the Aurel RFTide(TM) modules. These libraries provide functions to help you program an Arduino-controlled RFTide(TM) modules, and the included example projects demonstrate how to use.
RFTide(TM) is a 868MHz Aurel modules for wireless mesh network, developed for home and building automation.
For more information about the Aurel RFTide(TM) modules, please see http://www.rftide.com/.

Getting Started
Software
Download the archive from GitHub, decompress it, and move each library folder into the “libraries” subdirectory inside your Arduino sketchbook directory. You can view your sketchbook location by selecting File-Preferences in the Arduino
environment; if there is not already a “libraries” folder in that location, you should create it yourself. 
For more library installation info please visit  http://arduino.cc/en/Guide/Libraries. 
After installing the library, restart the Arduino environment so it can find the RFtide shield libraries and their examples.

Hardware
The RFTide shield for Arduino can be purchased on open-electronics website: http://www.open-electronics.org/
